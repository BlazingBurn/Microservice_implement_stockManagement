# Inventory-Manager
An inventory management system using Flask

### Prerequisites

To run this system you will need :

- Python 3
- Flask
- SQLALCHEMY
- WTForms

Assuming you have Python, proceed to install the rest using the command below:

```
pip3 install -r requirements.txt
```

# Built using
- Flask
- SQLAlchemy
